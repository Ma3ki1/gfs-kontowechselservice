# GFS Kontowechselservice — KI-Prototyp

Interaktiver Prototyp des KI-gestützten Kontowechselservices für
**Global Finance Solutions SE**, entwickelt von **metafinanz**.

## Setup & Start

```bash
pip3 install -r requirements.txt
streamlit run app.py
```

Öffnet automatisch unter `http://localhost:8501`.

## Demo-Modus

Über den Sidebar-Toggle "Demo-Modus" werden alle Schritte mit
realistischen Testdaten vorausgefüllt — ideal für die SteerCo-Präsentation
am 09.06.2026.
